**Дисциплина**: конструирование программного обеспечения
**Язык программирования**: C#
**Фреймворк**: Windows Forms

Современные торговые предприятия, в том числе магазины, специализирующиеся на продаже офисной техники, не могут работать без автоматизации процессов обслуживания клиентов и учета товаров.   

Ручное ведение документации и традиционные методы работы с заказами   
и складскими остатками стали малоэффективными, так как требуют больших временных затрат, повышают вероятность ошибок и снижают общую производительность труда.

Автоматизированное рабочее место (АРМ) продавца позволяет оптимизировать деятельность магазина за счёт использования программного обеспечения, которое дает быстрый доступ к информации о товаре, оформление продаж, ведение учета и формирование отчетности.   
Такое решение способствует повышению качества обслуживания клиентов   
и снижению нагрузки на сотрудников.

Целью разработки данного курсового проекта является повышение эффективности работы продавца магазина офисной техники за счёт внедрения АРМ, обеспечивающего быстрое и точное выполнение операций, связанных   
с обслуживанием покупателей, ведением учёта товаров и формированием отчетности.

**Основные задачи курсового проекта были поставлены следующие**:
	− обеспечение удобного поиска и просмотра ассортимента товаров;
	− автоматизация процесса оформления продаж;
	− учет движения товаров (поступление, реализация, остатки);
	− формирование отчетов для анализа деятельности магазина;
	− повышение эффективности работы продавца и сокращение времени обслуживания клиентов.
